## 1. 관리자 로그인

### 접속 방법
1. 브라우저에서 https://webinars.co.kr/admin/login 접속
2. 등록된 이메일과 비밀번호 입력
3. "로그인" 버튼 클릭

### 로그인 정보
- **URL**: https://webinars.co.kr/admin/login
- **이메일**: 관리자 이메일 주소
- **비밀번호**: 설정된 비밀번호

### 비밀번호 분실 시
1. 로그인 페이지에서 "비밀번호를 잊으셨나요?" 클릭
2. 등록된 이메일 주소 입력
3. 이메일로 전송된 재설정 링크 클릭
4. 새 비밀번호 설정

## 2. 포스트 작성

### 새 포스트 작성하기
1. 관리자 로그인 후 "새 글 작성" 버튼 클릭
2. 필수 정보 입력:
   - **제목**: 포스트 제목 (필수)
   - **슬러그**: URL에 사용될 주소 (자동 생성, 수정 가능)
   - **요약**: 목록에 표시될 짧은 설명
   - **본문**: Markdown 형식으로 작성

### 포스트 상태
| 상태 | 설명 |
|------|------|
| draft | 임시저장 상태, 공개되지 않음 |
| published | 발행됨, 모든 방문자에게 공개 |
| scheduled | 예약 발행 대기 중 |

### 발행 방법
1. **즉시 발행**: "발행" 버튼 클릭
2. **임시저장**: "저장" 버튼 클릭 (draft 상태 유지)
3. **예약 발행**: 예약 발행 일시 설정 후 저장

## 3. 포스트 관리

### 포스트 목록
- **URL**: https://webinars.co.kr/admin/posts
- 모든 포스트 목록 확인 가능
- 상태별 필터링 (전체/발행됨/임시저장)

### 포스트 수정
1. 포스트 목록에서 수정할 포스트 클릭
2. 내용 수정 후 "저장" 또는 "발행" 클릭

### 포스트 삭제
1. 포스트 목록에서 삭제할 포스트 선택
2. "삭제" 버튼 클릭
3. 확인 대화상자에서 "삭제" 선택

> **참고**: 삭제된 포스트는 소프트 삭제되어 데이터베이스에 보관됩니다.

### 포스트 비발행
- 발행된 포스트를 다시 비공개로 전환할 수 있습니다.
- 수정 페이지에서 "비발행" 버튼 클릭

## 4. 카테고리 및 태그

### 카테고리
- 포스트를 주제별로 분류
- 하나의 포스트에 하나의 카테고리 지정
- 예: 웨비나, 하이브리드 이벤트, 라이브 스트리밍, 인사이트

### 태그
- 포스트에 관련 키워드 추가
- 하나의 포스트에 여러 태그 지정 가능
- 예: #트렌드, #팁, #사례연구, #기술

### 카테고리/태그 활용
- 방문자가 관심 주제의 포스트를 쉽게 찾을 수 있음
- URL 예시:
  - 카테고리: `/blog/category/webinar`
  - 태그: `/blog/tag/trend`

## 5. 이미지 업로드

### 지원 형식
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- GIF (.gif)

### 파일 크기 제한
- 최대 5MB

### 업로드 방법
1. 에디터에서 이미지 버튼 클릭
2. 파일 선택 또는 드래그 앤 드롭
3. 업로드 완료 후 자동으로 본문에 삽입

### 대표 이미지 (Featured Image)
- 포스트 목록과 소셜 미디어 공유 시 표시
- 권장 크기: 1200 x 630px
- 별도의 "대표 이미지" 필드에 업로드

### 이미지 최적화 팁
1. WebP 형식 사용 권장 (용량 절감)
2. 적절한 크기로 리사이즈 후 업로드
3. 의미 있는 파일명 사용 (SEO 도움)

## 6. Markdown 작성법

### 기본 문법

#### 제목
```markdown
# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
#### 제목 4 (H4)
```

#### 텍스트 스타일
```markdown
**굵은 글씨**
*기울임*
~~취소선~~
`인라인 코드`
```

#### 목록
```markdown
- 순서 없는 목록 1
- 순서 없는 목록 2
  - 중첩 목록

1. 순서 있는 목록 1
2. 순서 있는 목록 2
```

#### 링크와 이미지
```markdown
[링크 텍스트](https://example.com)
![이미지 설명](이미지URL)
```

#### 인용문
```markdown
> 인용문 내용입니다.
> 여러 줄도 가능합니다.
```

#### 코드 블록
````markdown
```javascript
const greeting = "Hello, World!";
console.log(greeting);
```
````

#### 표
```markdown
| 항목 | 설명 |
|------|------|
| 항목1 | 설명1 |
| 항목2 | 설명2 |
```

#### 구분선
```markdown
---
```

### 작성 팁
1. 제목 계층 구조를 명확하게 유지
2. 단락 사이에 빈 줄 추가
3. 이미지에는 항상 설명(alt text) 추가
4. 긴 문장보다 짧은 문장 선호

## 7. SEO 최적화

### 제목 작성
- 핵심 키워드를 앞쪽에 배치
- 60자 이내 권장
- 숫자나 질문형 제목이 효과적

**좋은 예시:**
- "2024년 하이브리드 이벤트 트렌드 5가지"
- "라이브 스트리밍 품질을 높이는 방법"

### 요약(Excerpt) 작성
- 150-160자 이내
- 포스트 핵심 내용 요약
- 클릭을 유도하는 문구 포함

### 슬러그(URL) 설정
- 짧고 명확하게
- 한글보다 영문 권장
- 단어는 하이픈(-)으로 구분

**예시:**
- 좋음: `/blog/hybrid-event-trends-2024`
- 나쁨: `/blog/2024년-하이브리드-이벤트-트렌드-분석-및-전망`

### 이미지 SEO
- 파일명에 키워드 포함
- alt 텍스트 반드시 작성
- 적절한 파일 크기 유지

### 내부 링크
- 관련 포스트 간 링크 연결
- 앵커 텍스트에 키워드 포함

## 8. 문제 해결

### 로그인이 안 될 때
1. 이메일 주소 확인
2. 비밀번호 대소문자 확인
3. 브라우저 캐시/쿠키 삭제 후 재시도
4. 비밀번호 재설정 시도

### 이미지 업로드 실패
1. 파일 크기 확인 (5MB 이하)
2. 파일 형식 확인 (jpg, png, webp, gif)
3. 인터넷 연결 상태 확인
4. 다른 브라우저에서 시도

### 포스트가 보이지 않을 때
1. 포스트 상태가 "published"인지 확인
2. 삭제되지 않았는지 확인
3. 브라우저 캐시 삭제 후 새로고침

### 에디터 오류
1. 페이지 새로고침
2. 브라우저 캐시 삭제
3. 다른 브라우저에서 시도
4. 작성 중인 내용 백업 후 재접속

## 부록: 자주 묻는 질문 (FAQ)

### Q: 예약 발행은 어떻게 하나요?
A: 포스트 작성 페이지에서 "예약 발행" 일시를 설정하고 저장하면 됩니다. 지정된 시간에 자동으로 발행됩니다.

### Q: 포스트를 복사할 수 있나요?
A: 현재 복사 기능은 지원하지 않습니다. 새 포스트를 작성해주세요.

### Q: 여러 명이 동시에 편집할 수 있나요?
A: 동시 편집은 지원하지 않습니다. 마지막 저장 내용이 유지됩니다.

### Q: 포스트 조회수는 어디서 확인하나요?
A: 관리자 포스트 목록에서 각 포스트의 조회수를 확인할 수 있습니다.

### Q: 삭제한 포스트를 복구할 수 있나요?
A: 삭제된 포스트는 데이터베이스에 보관되어 있어 복구 가능합니다. 관리자에게 문의해주세요.
